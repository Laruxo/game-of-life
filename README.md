# Game of Life

A simple [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) implementation in React.

## Features

- Play/Pause the game
- Changing grid size
- Changing game speed
- Drawing on the grid
- Saving/Loading grid

## Running the project

- Run dev environment
```
npm run start
```

- Build app for production
```
npm run build
```

- Lint code with ESLint
```
npm run lint
```

- Run tests
```
npm run test
```
